4. Author details (name, event, reputation score, gold badge, silver badge and bronze badge) 		Have logged all the answers on which the user spends more than 3 secs which shows the answer is useful to the user or difficult to understand
Snippet click 1. Title
2. Answer / Question
3. Code
4. Author details (name, event, reputation score, gold badge, silver badge and bronze badge) 		When the user clicks run snippet code button the corresponding post is logged which can be used to find snippets that has been run by more users
Detect copy event 1. Title
2. Answer / Question
3. Copied text
4. Author details (name, event, reputation score, gold badge, silver badge and bronze badge) 		When the user copies snippet / code from the answers posted by other users the answer details and author details are logged. This can be used to find which answer is copied more and credits can be given to those answers / authors
Vote up / down 1. Title
2. Answer / Question
3. Code
4. Author details (name, event, reputation score, gold badge, silver badge and bronze badge) 		Have logged the posts in which user has voted along with author details so that it can be used to find which user's post has been voted the most by our user and credits can be given to the author based on that
Scroll down 1. Post
2. Scroll Position
User's scroll event is captured when the user reaches 90% of the page. This may show that the user didn't find the answer till the end of page
Search 1. Search text User's search event is captured along with the search criteria which can be used to show related links at top in future based on the search
Bookmark page 1. Bookmark title
2. URL 		Have logged the page title and link which the user has bookmarked so that user interested pages are logged
Page load 1. Post
2. Answers count 		All the pages visited by user is logged with post details and answers count. This can be used to track views fo the post
Post Links in summary 1. Link text
2. Answers count
3. Vote count
4. Views count
5. Current page number
5. Source post
6. Source link
7. Link
8. Serial number
9. Related tags 		All links which are displayed in summary and clicked by user are logged. Attributes like answers count, vote count and views count help us to rate the post link. Current page number helps us to identify in which page the user has found useful link. Serial number is used to identify the useful link within a page. These can be used to personalize search results for the user in future. Also the source page from which this link was clicked is tracked so that the sequence of pages visited by the user can be tracked.

2. Question / Answer 		Have logged when the user posts questions / answers which can be used to find the topics the user is interested in

2. Question

Related tags click 1. Title
2. Page number 		The related tags which the user has clicked has been logged so that user interested tags can be identified

2. Answer / Question

This can be used to personalize search for the user by showing results posted by user which has been shared more. The edit details can be used to identify post which has undergone multiple edits
Page visited 1. Title
2. Page number 		The pages visited by user is tracked which can be used to improve the search results and show relevant links in first few pages
